# Test result for Scentronix fullstack position

## Backend

Require: node version greater than v22

Setup

`yarn install`
or
`npm install`

Run test

`yarn test`
or
`npm run test`

## Frontend

Require: node version greater than v22

Setup

`yarn install`
or
`npm install`

Run dev

`yarn dev`
or
`npm run dev`
